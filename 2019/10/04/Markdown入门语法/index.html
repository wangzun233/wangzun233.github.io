Markdown入门语法
发表于 2019-10-04| 更新于 2019-10-04|学习日记
阅读量:
  1. 了解Markdown
    • 1.1 人们在网上发布的文章大多都是通过 富文本编辑器 将所写内容+排版转换为Html格式，所显示出来。
      而如今有了Markdown，人们能够更加方便地对文章进行排版修饰。
    • 1.2 Markdown 是一种常见的 标记语言 (非编程语言)，它能帮助我们 快速排版文档。

  1. Markdown常用用法

    • 2.1标题

      # 这是一级标题
## 这是二级标题
### 这是三级标题
#### 这是四级标题
##### 这是五级标题
###### 这是六级标题

    • 2.2换行
      Markdown有两种换行方式：

      • 1.使用html中的换行标签<br>
      • 2.两个回车

    • 2.3分割线
      Markdown有两种分割方式：

      • 1.使用html中的分割线标签<hr>
      • 2.***

    • 2.4 引用
      Markdown中引用直接使用 >（大于号）即可 >

    • 2.5 转义
      当我们在符号前加上 \ 它就会转义，不会产生应有效果

      • #这是一级标题
      • >引用

    • 2.6 常见字体效果
      在Markdown中有4种常见字体效果：

      这是加粗的文字
      这是倾斜的文字`
      这是斜体加粗的文字
      这是加删除线的文字

        **这是加粗的文字**
  *这是倾斜的文字*`
  ***这是斜体加粗的文字***
  ~~这是加删除线的文字~~

    • 2.7 待办事项

      • 未完成事项
      • 已完成事项
        ```
      • 未完成事项
      • 已完成事项
        ```

    • 2.8 超链接 & 图片
      Markdown中 超链接 和 显示网页图片 其实差不多，就一个 ! 的区别

      [博客](wangzun233.top)
![image](http://img.wangzun233.top/IMG_20190609_111212-01.jpeg)

    • 2.9 代码高亮
      编程语言 & key 对应表（常用语言）

       ```key
 code

    • 2.11 列表
      在Markdown中有两种列表：无序列表（无数字）、有序列表（有数字）

      - 列表1(无序列表)
 - 列表1.1
 - 列表1.2
1. 列表1(有序列表)
 - 内容1.1
2. 列表2
3. 列表3

    • 2.11 表格

      表头1 表头2 表头3
      内容1 内容2 内容3
      内容4 内容5 内容6 
      表头1 |表头2|表头3
      -----|-----|-----
      内容1|内容2|内容3
      内容4|内容5|内容6
