Trie树

维护一个字符串集合，支持两种操作：

  1. “I x”向集合中插入一个字符串x；
  2. “Q x”询问一个字符串在集合中出现了多少次。

共有N个操作，输入的字符串总长度不超过 105105，字符串仅包含小写英文字母。

输入格式

第一行包含整数N，表示操作数。

接下来N行，每行包含一个操作指令，指令为”I x”或”Q x”中的一种。

输出格式

对于每个询问指令”Q x”，都要输出一个整数作为结果，表示x在集合中出现的次数。

每个结果占一行。

数据范围

1≤N≤2∗1041≤N≤2∗104

输入样例：

5
I abc
Q abc
Q ab
I ab
Q ab

输出样例：

1
0
1

Trie树原理：

题解：

//Trie树 高效的存储和查找字符串集合 的数据机构
//Trie树的实现方式见图
#include <iostream>
using namespace std;
const int N = 1e4+10;
//son[值][边]  cnt单词节点 idx当前用到那个下标；下标是0的节点既是根节点又是空节点
int son[N][26],cnt[N],idx;
void insert(char str[])
{
    int p = 0; //字符串最后以零结尾
    for(int i=0;str[i];i++)
    {
        int u = str[i] - 'a';//将字母映射成数字
        if(!son[p][u]) son[p][u] = ++idx;//如果字母不存在增加节点
        p = son[p][u];//最后把单词节点赋值过去
    }
    cnt[p] ++;//单词节点++
}
int query(char str[])
{
    int p =0;
    for(int i=0;str[i];i++)
    {
        int u = str[i] - 'a';
        if(!son[p][u]) return 0;//如果不存在查询的节点，直接返回
        p = son[p][u];
    }
    return cnt[p];
}
int main()
{
    cin.tie(0);
    ios :: sync_with_stdio(false);
    int n;
    cin >> n;
    while(n--)
    {
        char l;
        char x[N];
        cin >> l;
        if(l == 'I')
        {
            cin >> x;
            insert(x);
        }
        else
        {
            cin >> x;
            int s =query(x);
            cout << s << endl;
        }
    }
    return 0;
}
